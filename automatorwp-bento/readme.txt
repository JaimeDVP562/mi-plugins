== Description ==

[Bento](https://www.bentonow.com/ "Bento") is an email and onsite marketing automation platform for ecommerce brands.

= Actions =

* Add a tag to a subscriber.
* Remove a tag from a subscriber.
* Create or update a subscriber.
* Remove a subscriber.
* Add a subscriber to a campaign.
