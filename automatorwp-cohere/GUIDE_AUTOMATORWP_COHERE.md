## Introducción

Esta guía recopila los pasos de prueba realizados sobre el plugin AutomatorWP Cohere y explica cómo aprovechar al 
máximo las acciones disponibles, adaptando los prompts y categorías para cubrir casos avanzados como análisis de 
sentimiento, detección de temas, moderación, extracción de entidades, detección de idioma, etc.

Para obtener las credenciales de la API de Cohere, visita [Cohere AI](https://cohere.ai/), create una cuenta y genera tu API Key. 
Luego, ingresa esta clave en la configuración del plugin AutomatorWP Cohere y comprueba la conexión pulsando el botón 
de Authorize y comprueba que las credenciales son correctas. Una vez configurado, puedes utilizar las acciones 
disponibles.
